# Part 2: Distributed Sedona - Big Data Geospatial Processing
## 10-Hour Course

Scale beyond single-node limitations with Apache Sedona for distributed geospatial processing across clusters.

---

## Prerequisites

Complete **[Part 1: SedonaDB](../part1_sedonadb/README.md)** or have equivalent experience with:
- Spatial SQL and geometry operations
- Basic understanding of geospatial concepts
- Python or Scala programming

**Additional:** Basic Apache Spark knowledge recommended

## What You'll Learn

- Process billions of geometries across distributed clusters
- Perform spatial queries and joins at scale
- Optimize with spatial indexing and partitioning
- Build real-time spatial analytics pipelines
- Deploy production distributed applications
- Apply best practices from industry use cases

---

## When to Use Distributed Sedona

- Data > 100GB
- Processing billions of geometries
- Real-time streaming required
- Horizontal scaling needed
- Integration with Spark ecosystem
