## Final Project: Urban Services Dashboard

Build a complete urban services analysis tool that:
- Loads city infrastructure data (schools, hospitals, parks)
- Analyzes service coverage by neighborhood
- Identifies underserved areas
- Provides web API for queries
- Visualizes results on interactive map

**Deliverables:**
- SedonaDB database with spatial indexes
- Python API (FastAPI)
- Interactive web map (Folium)
- Analysis report

**Duration:** 3-4 hours
